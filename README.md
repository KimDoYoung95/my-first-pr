# my-first-pr

A simple Python calculator project.

## Features

- Add two numbers
- Subtract two numbers
- Multiply two numbers
- Divide two numbers

## Usage

```python
from calculator import add, subtract, multiply, divide

print(add(2, 3))       # 5
print(subtract(10, 4)) # 6
print(multiply(3, 4))  # 12
print(divide(10, 2))   # 5.0
```

## Requirements

- Python 3.8+
